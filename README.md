# my-first-code
My first code :) a little bank

A customer has a balance of $100 and is asked:

"How much money would you like to withdraw?"

If the customer wants to withdraw $50, the program checks whether there is enough money available in the account balance.

If sufficient funds are available, $50 is deducted from the balance and the following message is displayed:

"You have successfully withdrawn $50! Current account balance: $50"

If the customer tries to withdraw $120, no money is deducted from the balance because the account does not contain enough funds.

The following message is displayed:

"You do not have enough funds! Your current account balance is $100"
